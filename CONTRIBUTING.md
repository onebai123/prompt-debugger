# 贡献指南

感谢你对 Prompt Debugger 项目的关注！我们欢迎所有形式的贡献。

## 如何贡献

### 报告问题
- 在 GitHub Issues 中提交 Bug 报告
- 提供详细的复现步骤
- 附上截图或错误日志

### 提交功能建议
- 在 GitHub Issues 中描述你的想法
- 说明这个功能的使用场景
- 讨论可能的实现方案

### 提交代码
1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 提交 Pull Request

## 开发指南

### 环境要求
- Node.js 18+
- npm 或 yarn

### 本地开发
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建
npm run build
```

### 代码规范
- 使用 TypeScript
- 遵循 ESLint 规则
- 保持代码简洁清晰

### 提交规范
- feat: 新功能
- fix: Bug修复
- docs: 文档更新
- style: 代码格式调整
- refactor: 代码重构
- test: 测试相关
- chore: 构建/工具链更新

## 行为准则
- 尊重所有贡献者
- 保持友好和专业
- 欢迎建设性的批评

## 许可证
贡献的代码将采用 MIT 许可证。
